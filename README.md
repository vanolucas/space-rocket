# Space Rocket

A Web toy where you control a rocket with your keyboard.

## Keyboard controls

- **Up/Down arrows**: forward/backward thrust.
- **Left/Right arrows**: rotate.
- **W/Z, A/Q, S, D**: move up, left, down, right.
- **Space**: shoot bullet.
